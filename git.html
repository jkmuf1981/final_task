Git

  • git init-создание нового репозитория Git в пустой директорий
  • git clone -клонирование репозитория из удалённого хранилища на локальную машину
  • git push-выгрузка сделанных изменений в удалённый репозиторий
  • git pull-получение изменений из удалённого репозитория на локальную
  • git status -просмотр списка измененных файлов
  • git branch -создание ветки
  • git commit - сохранение изменений в git
  • git log -посмотреть истории коммитов
